# SDG 8 Demo Site

Simple static demo showing a problem related to SDG 8 (Decent Work & Economic Growth).

How to run

1. Open `index.html` in your browser (no server required):

```bash
open index.html
```

What it does

- Shows quick context about youth unemployment.
- A simple job-creation calculator: estimate jobs from investment and cost-per-job.
- Collects community proposals stored in `localStorage`.

Next steps (optional):

- Add charts or region-specific data.
- Connect proposals to a backend to persist and moderate entries.
- Localize content and provide multilingual support.
